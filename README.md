# 🎓 Student Management System

## 🚀 Features Implemented

### ✅ Student Module
- Create Student
- Edit Student
- Delete Student
- View Details
- Form Validation

### ✅ Course Module
- Multiple Course Selection
- Checkbox Binding

### ✅ Relationships
- Many-to-Many Relationship
- Navigation Properties
